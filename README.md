# Mini SOC Lab with Wazuh SIEM

## Overview

This project demonstrates the implementation of a Mini Security Operations Center (SOC) using Wazuh SIEM in a virtualized lab environment. The objective is to collect, monitor, analyze, and respond to security events generated by Windows and Linux endpoints.

The lab simulates a small enterprise network where endpoint logs are forwarded to a centralized Wazuh server for security monitoring and threat detection.

---

## Objectives

* Deploy Wazuh SIEM in a virtual environment.
* Monitor Windows and Linux endpoints.
* Detect suspicious activities and security events.
* Perform File Integrity Monitoring (FIM).
* Conduct Vulnerability Assessment.
* Map alerts to MITRE ATT&CK techniques.
* Demonstrate basic SOC analyst workflows.
